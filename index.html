Mathematical Roses

This a rose drawing application. What is a rose? A rose is the set of points in polar coordinates specified by the polar equation.

Polar Equations:

x = r * cos(k * a)
y = r * sin(k * b)
k = n/d

When you run the program, you can change the n and d values to draw any shape from the picture below. Press n to increase the red values and d to increase the blue values.

For more information, you can check the references.
